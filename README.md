# MCL-Votifier-OG

A soft fork of [MCL-Votifier](https://github.com/kaziu687/MCL-Votifier), a plugin for Minecraft Bukkit/Spigot servers that let your users gain rewards for votes at https://minecraft-servers.gg

Maintained by and for [TrueOG Network](https://true-og.net/).

## Changes over MCL-Votifier:

- Support 1.19.4.
- Update gradle to 8.8.
- Replace deprecated code.

More informations about installation can be found at:
https://minecraft-servers.gg/mcl-votifier-plugin
